#Style Guide

- Tabs, not spaces

- Naming conventions
    - Class names `PascalCase`
    - Method names `camelCase`
    - Variable names `camelCase`
 
- New line at the end of all files

- Same line brackets, e.g:
    ``` 
    void func() {
        //code
    }
    ```
    not
    ``` 
    void func()
    {
        //code
    }
    ```
